# Smart Academic Calendar (MERN)

A smart, dynamic academic calendar application designed to help students and educators manage schedules, assignments, exams, and institutional events efficiently. Built using the MERN stack (MongoDB, Express.js, React, Node.js).

---

## 🚀 Features

* **Interactive Calendar View:** Daily, weekly, and monthly views for classes, exams, and deadlines.
* **Smart Reminders:** Automated notifications or visual alerts for upcoming academic milestones.
* **Task Management:** Integrated To-Do lists paired with course deadlines.
* **Course Categorization:** Color-coded events based on subjects or course codes.
* **User Authentication:** Secure registration and login for students and faculty.

---

## 🛠️ Tech Stack

**Frontend:**
* React.js
* Tailwind CSS / Material-UI (or specify your styling choice)
* Axios (for API requests)

**Backend:**
* Node.js & Express.js
* RESTful API architecture

**Database & Auth:**
* MongoDB & Mongoose ODM
* JSON Web Tokens (JWT) for secure authentication
